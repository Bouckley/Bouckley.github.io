# Jordan Bouckley Portfolio

Personal portfolio website showcasing my projects, skills, and experience as a Computer Science student at Queen's University.

## Technologies

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- Framer Motion

## Getting Started

```sh
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Features

- Responsive design
- Dark/Light theme toggle
- Smooth animations and transitions
- Interactive project showcase
- Contact form integration
- SEO optimized
